Heatify [BETA]

Heatify tracks your git repositories from Jazz and GitHub, and crunches the latest commit statistics for you to view in a HeatMap.

The goal of Heatify is to encourage good Agile development practices. That is, we would like to encourage developers to contribute small, coherent commits, frequently. Commits should precisely address one issue or feature within the project.

Getting Started
Heatify works out-of-the-box on public Jazz and GitHub git repositories; simply copy and paste the http URL into the field below to start tracking and view the HeatMap! To view a HeatMap for a private repository, first add the Git-Monitor Bot as a member to your repo.
